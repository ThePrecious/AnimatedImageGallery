#Animated Image Gallery - HTML5
 This is a HTML5 Animated Image gallery, with jQuery - quicksand plugin for animation, and fancybox for displaying the image.
 
 On selecting a category from the nav bar, the quicksand plugin shows only those images that belong to the selected category, with some animation. On clicking an image, fancybox is used to display the image in a pop up window.
